# Todo

> Simple todo app to manage your tasks. Application was written in JavaScript (ES6)

[See demo](https://fncoder.github.io/todo/)

## Features

- Add new items
- Bookmarks as All, Complete, Incomplete with filters
- Remove items
- Simple design
